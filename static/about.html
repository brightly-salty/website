A quizbowl packet reader created by Geoffrey Wu, with contributions from Brian Lai.

Packets are collected from quizbowlpackets.com and parsed into JSON files using a Python-based quizbowl packet parser, which I built after being dissatisfied with YAPP (due to its strict formatting requirements and lack of category support). If you notice any issues with packets or categorization, please contact me. Packet numbers are numbered starting from 1 based on the following order: numbered packets (1, 2, 3, etc. or A, B, C, etc.), then editor questions, then non-numbered finals packets, and then the remaining packets are numbered in alphabetical order.

Feel free to check out the source code for this website and contribute!

Why QB Reader?

  • Only quizbowl reader to use the latest set of questions.
    • As far as I am aware, all other quizbowl reading software (including Discord bots) pull their questions off of the quizdb database, which is not up to date.
  • Only reader that allows you to choose questions by packet (quizbug technically allows you to select by set, but that's not an intended feature and the SSL certificate is expired).
  • Excellent support for mobile.
  • Strong support for categories and subcategories.
  • Ability to read both tossups and bonuses.
Currently, the database contains:
  • 138 sets.
  • 1961 packets.
  • 40288 tossups.
  • 36724 bonuses.

Planned Features:

  • Multiplayer support.
  • AI that plays against you with an adjustable difficulty.
  • NHBB (National History Bee and Bowl) sets.
