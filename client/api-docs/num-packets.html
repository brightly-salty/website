GET qbreader.org/api/num-packets

Get the number of packets in a set.

Parameters
  • setName: string required
    The name of the set. See /set-list for a list of all sets.

Returns
A JSON object with the following properties:
  • numPackets: number
    The number of packets in the set. If a set with the given name does not exist, this will be 0, and a 404 error will be returned.
