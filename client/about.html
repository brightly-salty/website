A quizbowl packet reader created by Geoffrey Wu, with contributions from Brian Lai.

Packets are collected from quizbowlpackets.com and parsed into JSON files using a Python-based quizbowl packet parser. If you notice any issues with packets or categorization, please contact me. Packet numbers are numbered starting from 1 based on the following order: numbered packets (1, 2, 3, etc. or A, B, C, etc.), then editor questions, then non-numbered finals packets, and then the remaining packets are numbered in alphabetical order.

Feel free to contribute to the source code.

Why QB Reader?

  • Uses the latest set of questions. No more weirdly formatted questions from the 1990s.
  • Allows you to choose questions by packet.
  • Excellent mobile support.
  • Strong support for categories and subcategories.
  • Ability to read both tossups and bonuses.
  • Over 100,000 questions from over 170 sets.

Planned Features:

  • AI that plays against you with an adjustable difficulty.
  • NHBB (National History Bee and Bowl) sets.
