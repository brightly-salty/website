A text-based quizbowl packet reader created by Geoffrey Wu,

with contributions from Brian Lai and Ryan Rosenberg.

Packets are collected from quizbowlpackets.com and parsed into JSON files using a Python-based quizbowl packet parser. If you notice any issues with packets or categorization, please report the question or contact me on discord at thedoge#1189. Packets are numbered by alphabetical order of their names.

Feel free to contribute to the source code.

Why QB Reader?

  • Uses the latest set of questions. No more weirdly formatted questions from the 1990s.
  • Allows you to filter questions by tossup/bonus, difficulty, packet, and category/subcategory.
  • Excellent mobile support.
  • 167,059 questions from 310 sets.
